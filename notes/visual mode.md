# Visual mode


## Visual mode

[Vim 101 visual mode](http://usevim.com/2012/05/11/visual)

[vimdoc visual](http://vimdoc.sourceforge.net/htmldoc/visual.html)

    v # per character
    V # linewise
    <C-v) # blockwise
    
    # select4 lines above
    4k
    
    # select 3 lines below
    3j
    
    # select 2 words
    2w
    
    # go back to the previous visual mode
    gv
    
    # go back to the other end
    o # or O
    
    # operators help
    :h visual-operators
